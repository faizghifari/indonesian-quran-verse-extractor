REPUBLIKA.CO.ID, BEIJING -- Seorang pria, yang diduga membocorkan informasi data empat juta lebih pengguna laman resmi pemesanan tiket kereta api di Cina, ditangkap. Tim pengawal keamanan siber Beijing bersama petugas Kepolisian Distrik Xicheng menangkap pria bermarga Chen. Ia ditangkap atas dugaan menjual informasi data registrasi lebih dari 600 ribu orang serta informasi data 4,1 juta pengguna laman resmi pemesanan tiket kereta api 12306.cn, demikian dilaporkan media resmi setempat, Rabu (2/1). Manajemen China Railway pada 28 Desember 2018 menyatakan bahwa 12306.cn tidak membocorkan informasi data pengguna. Meskipun demikian, pihaknya tetap mengingatkan kepada para pengguna laman pemesanan tiket kereta api tersebut agar berhati-hati. Dalam investigasinya, pihak kepolisian mendapati Chen secara ilegal membeli informasi data pengguna saat mendaftar, namun pelaku tidak mencuri. Dia menggunakan data pendaftar baru tersebut untuk memperoleh informasi data 4,1 juta pengguna melalui sarana pihak ketiga dalam pemesanan tiket. Polisi juga mendapati Chen (25 tahun), yang berasal dari Provinsi Hebei, bekerja di salah satu perusahaan teknologi yang berlokasi di Beijing.
